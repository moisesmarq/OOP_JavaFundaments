# POO Java Fundamentos

## Objetivo


> O intuito desse projeto é colocar em prática os conceitos aprendidos ao decorrer do curso, o qual é ministrado pelo professor Nélio Alves.


### Principais conteúdos:

    - Introdução à Programação Orientada a Objetos
    - Construtores, this, sobrecarga, encapsulamento
    - Comportamento de memória, arrays, listas
    - Tópicos especiais em Java: data-hora
    - Enumerações e composição
    - Herança e polimorfismo
    - Tratamento de exceções
    - Trabalhando com arquivos
    - Interfaces
    - Generics, Set, Map
    - Programação funcional e expressões lambda****
